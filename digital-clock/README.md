# डिजिटल घड़ी - मल्टी टाइम जोन

## विवरण

यह एक आधुनिक वेब-आधारित डिजिटल घड़ी एप्लिकेशन है जो विश्व के विभिन्न समय क्षेत्रों को प्रदर्शित करता है।

## विशेषताएं

### 🕐 मुख्य विशेषताएं

1. **डिजिटल समय प्रदर्शन**
   - वर्तमान समय बड़े अक्षरों में
   - तारीख हिंदी में
   - समय क्षेत्र की जानकारी

2. **मल्टी टाइम जोन समर्थन**
   - 18 प्रमुख विश्व समय क्षेत्र
   - Clock cards में सभी समय क्षेत्र
   - UTC ऑफसेट दिखाता है
   - समय क्षेत्र को जोड़ें/हटाएं

3. **एनालॉग घड़ी**
   - वर्तमान भारतीय समय का एनालॉग प्रदर्शन
   - घंटे, मिनट और सेकंड की सुई
   - 12 घंटे का प्रारूप

4. **प्रारूप विकल्प**
   - 12 घंटे / 24 घंटे का प्रारूप
   - सेकंड दिखाने/छिपाने का विकल्प

5. **अलर्ट प्रणाली**
   - कस्टम समय पर अलर्ट सेट करें
   - अलर्ट का नाम दें
   - ध्वनि सूचना
   - अलर्ट की सूची देखें

6. **थीम विकल्प**
   - लाइट मोड (डिफॉल्ट)
   - डार्क मोड
   - स्मूथ एनिमेशन

7. **अतिरिक्त विकल्प**
   - ध्वनि सूचना सक्षम/अक्षम करें
   - एनालॉग घड़ी दिखाएं/छिपाएं
   - LocalStorage में सेटिंग्स सहेजी जाती हैं

## उपयोग कैसे करें

### 1. समय क्षेत्र जोड़ें
```
1. "समय क्षेत्र जोड़ें" सेक्शन में ड्रॉपडाउन से चुनें
2. "जोड़ें" बटन पर क्लिक करें
3. नया clock card दिखाई देगा
```

### 2. अलर्ट सेट करें
```
1. "अलर्ट सेट करें" सेक्शन में समय चुनें
2. (वैकल्पिक) अलर्ट का नाम दें
3. "अलर्ट सेट करें" बटन पर क्लिक करें
```

### 3. सेटिंग्स बदलें
```
1. 24 घंटे का प्रारूप चालू/बंद करें
2. सेकंड दिखाएं/छिपाएं
3. एनालॉग घड़ी दिखाएं/छिपाएं
4. डार्क मोड चालू/बंद करें
```

## प्रमुख JavaScript APIs

1. **Intl.DateTimeFormat**
   - विभिन्न क्षेत्रों में तारीख/समय फॉर्मेट करने के लिए

2. **Canvas API**

3. **LocalStorage**
   - सेटिंग्स सहेजने के लिए

   - अलर्ट ध्वनि बजाने के लिए

## कुछ उपयोगी JavaScript कोड

### समय को किसी अन्य टाइम जोन में प्राप्त करें
```javascript
const timezone = 'Asia/Tokyo';
const now = new Date();
const timeInTZ = new Date(now.toLocaleString('en-US', { timeZone: timezone }));
```

### Intl API से तारीख फॉर्मेट करें
```javascript
const options = {
    weekday: 'long',
    year: 'numeric',
    month: 'long',
    day: 'numeric'
};
const formatted = new Intl.DateTimeFormat('hi-IN', options).format(date);
```

### LocalStorage में डेटा सहेजें
```javascript
const data = { clocks: ['Asia/Tokyo', 'America/New_York'] };
localStorage.setItem('clockSettings', JSON.stringify(data));
const loaded = JSON.parse(localStorage.getItem('clockSettings'));
```
